# Projeto IENA
Gerenciamento de frequências de membros nas celulas.
O sistema consiste em um cadastro de usuários, divididos em células, cadatrando membros as suas celulas.

# Get Started

__requisitos__
- Rails 5.0.0.1
- ruby 2.3.0p0
- database sqlite3
- gem 'sqlite3'
- gem 'puma', '~> 3.0'
- gem 'sass-rails', '~> 5.0'
- gem 'uglifier', '>= 1.3.0'
- gem 'coffee-rails', '~> 4.2'
- gem 'jquery-rails'
- gem 'turbolinks', '~> 5'
- gem 'jbuilder', '~> 2.5'
- gem 'bcrypt', '~> 3.1.7'
- bootstrap 
- jquery

# instalação
```
bundle install
```
